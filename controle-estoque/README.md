# Controle de Estoque

Sistema simples de controle de estoque usando Flask.

## Como rodar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute o app:
   ```bash
   python app.py
   ```

## Estrutura
- `app.py`: Backend Flask
- `requirements.txt`: Dependências
- `templates/`: HTML
- `static/css/`: Estilos
- `static/js/`: Scripts JS
